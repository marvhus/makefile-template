# Makefile Template
A C++ Makefile Template  
made by marvhus

## Quickstart

### Compile/Build
compiles/builds the C++ code in `src/impl/` and headers in `src/incl/` to `build/`
```sh
make build
```

### Run
runs the compiled binary in `build/`
```sh
make run
```

### Clean
removes the `build/` folder
```sh
make clean
```

### Multiple at once
You can run multiple ones at once, for example:
```sh
make build run
```
or:
```sh
make clean build
```
etc...
